# Changelog

## 0.1.0 — Initial Release

Opinionated oxlint plugin enforcing Effect-first development conventions.

54 rules covering statement bans, import bans, member expression bans, call expression rules, type-level rules, complex contextual rules, and pattern enforcement rules. All rules ship in the `recommended` category, so the entire rule set can be enabled with a single `"categories": { "recommended": "error" }` entry in `oxlint.json`.

See the [README](./README.md) for the full rule list with examples.
